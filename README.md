# 基于墨奇万象拼音的自然龙码
简称龙象般若
## 说明
基于rime平台的中文输入法解决方案

基于墨奇音形,替换了万象拼音的字词库,用上自然龙音码。
整合了墨奇万象方案的词库,合并细胞词库和腾讯词库并去重,保留了白霜词频。
由于合并后的词库太大,分成了几个小细胞库
用户可以自己选择开启

包里还有虎码单字方案,并增加了按"!"反查,用虎码打字时如果不记得怎么拆字了,可以按"!"然后输入龙象音形来查找该字的虎码

开启了＂，＂和＂．＂键翻页候选字，完全是个人习惯。
其实用了龙码和墨奇，候选字应该不用翻页啦
